# Void Survivors — CLAUDE.md

## 프로젝트 개요
도형 기반 2D 뱀서라이크 브라우저 게임.
차원의 균열에 빨려 들어간 존재가 파편을 모아 살아남는 게임.

## 기술 스택
- Phaser.js 3 (게임 엔진)
- HTML5 Canvas
- Vanilla JavaScript
- Supabase (로깅 전용)
- GitHub Pages (배포)

## 프로젝트 구조
void-survivors/
├── index.html
├── src/
│   ├── scenes/        → 게임 씬 (Boot, Game, UI, GameOver)
│   ├── entities/      → 플레이어, 적, 보스, 아이템
│   ├── systems/       → 웨이브, 아이템 조합, 레벨업
│   └── config/        → 게임 설정, 밸런스 수치
├── assets/            → 도형 에셋 (MVP는 Phaser 도형 사용)
└── CLAUDE.md

## 핵심 게임 루프
1. 준비 페이즈 (30초) → 맵에 흩어진 파편 아이템 줍기
2. 전투 페이즈 → 웨이브 클리어 + 보스
3. 라운드 클리어 → 아이템 강화 or 새 아이템 선택
4. 같은 계열 아이템 7개 → 극강 아이템 변환

## 코딩 규칙
- 모든 주석은 한국어로
- 파일 하나 최대 200줄
- 씬 간 데이터는 Phaser Registry 사용
- 밸런스 수치는 반드시 src/config/ 에서 관리
- 절대 하드코딩 금지

## 에이전트 공통 규칙
- 코드 수정 전 반드시 현재 구조 파악
- 기능 추가 시 config/ 수치 먼저 정의
- 버그 수정 시 원인 한 줄 주석으로 남기기
- MVP 범위 벗어나는 기능 제안 금지
